## Závislosti
* python3
* pdflatex

## Instalace
`git clone ￼https://github.com/nimral/nakup.git`

## Použití
V souboru `jidelnicek.py` je příklad definice jídelníčku. Pokud soubor spustíme v pythonu3 (`python3 jidelnicek.py`), vygeneruje se [nakupní seznam](seznam.pdf).
